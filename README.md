# Todo List Web Application

A simple full-stack Todo List application built using Python Flask and REST APIs.

## Features
- Add a todo
- View all todos
- Update todo status
- Delete a todo

## Tech Stack
- Python
- Flask
- SQLite
- HTML, CSS, JavaScript

## API Endpoints
- POST /todos
- GET /todos
- PUT /todos/<id>
- DELETE /todos/<id>

## Purpose
This project was built to apply REST API concepts learned through certification.
